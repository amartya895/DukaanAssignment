# Dashboard

## Overview

This is a simple dashboard project built with React, utilizing Tailwind CSS for styling. The project is organized into separate components for modularity and maintainability.

## Project Structure

- **src/**
  - **components/**
    - **NavBar**: Navigation bar component.
    - **PanelHeader**: Panel header component.
    - **Overview**: Overview component in panel.
    - **Pagination**: Pagiantion in panel  component.
    - **ViewBox**: overview inside component.
    - **Panel**: Panel component.
    - **NavOptions**:component. for nav options available.
    - **DataView**: Table content component.
  - **data/**
    - **navOptionSvg.js**: Data file for Navbar SVGs.
    - **orderDetails.json**: Data file for order details.
    - **panleSvg.js**: Data file for order details.
